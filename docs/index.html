TuyaRegisterWizard

A wrapper that combines @tuyapi/cloud and TuyaRegister.manual (included in this package) to make registration Just Work™️.

new TuyaRegisterWizard(options: Object)
Parameters
options (Object) construction options
Name Description
options.apiKey String API key
options.apiSecret String API secret
options.email String user email
options.password String user password
options.region String (default 'AZ') region (AZ=Americas, AY=Asia, EU=Europe)
options.timezone String (default '-05:00') timezone of device
Example
// Note: user account does not need to already exist
const register = new TuyaRegister.wizard({key: 'your-api-key',
                                          secret: 'your-api-secret',
                                          email: 'example@example.com',
                                          password: 'example-password'});
